# korge-dragonbones

Dragonbones plugin for KorGE using kproject.

## Usage

By using kproject:

### `deps.kproject.yml`

```kotlin
dependencies:
  - git::korge-dragonbones::korlibs/korge-dragonbones::/korge-dragonbones::da64d67195a7703a589d58f93cadfa3e00232685
```
